Tech

EM foloseste urmatoarele technologii:

  • node.js - JavaScript backend
  • electron - aplicatii pentru desktop folosind technologii web
  • reactjs - librarie pentru User Interfaces
  • Twitter Bootstrap - UI boilerplate
  • Gulp - streaming build system

Development

Vrei sa contribui? Foarte bine!

Cum sa contribui:

Instaleaza git, node si npm. Indiferent daca folosti OS X, Linux sau Windows, asigurate ca binarele sunt introduse in PATH. Deschide un terminal si executa:

$ git clone <repo>
$ cd em-electron
$ npm install
$ npm install gulp -g
$ gulp
